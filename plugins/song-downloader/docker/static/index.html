Song Downloader

Search by song name and artists, download the full audio song, optionally rename it, and auto-transfer it into /mnt/nas/media/music.

This app may use the Media Downloader API to fetch audio when that API is reachable. It falls back to direct yt-dlp when needed.

v1.0.1
Final filename format is expected to be <song name> - <comma separated artists>.mp3.
